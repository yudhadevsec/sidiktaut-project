# sidiktaut-project
Sidik taut merupakan tools keamanan siber untuk menganalisis dan mendeteksi tautan (URL) berbahaya, phishing atau malware secara realtime. Project ini memanfaatkan Threat Intelligence dari VirusTotal API untuk memberikan hasil yang akurat.
Dibangun tanpa Database untuk efiensi,fleksibelitas tinggi dan privasi pengguna.

Fitur Utama
Project ini terdiri dari 3 modul utama yang saling terintegrasi.
